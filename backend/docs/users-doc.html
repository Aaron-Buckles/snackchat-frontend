Users

/api/users

  • GET /api/users

    Get a list of all user names.

    Success Response

    [User.name]

    Error Response

    { err }

  • POST /api/users/signup

    Signup a user with name, email, and password.

    JSON Data

    • name: The name of the user (both first and last)
    • email: The email of the user (must be formatted correctly)
    • password: The user's password (will be encrypted)

    Success Response

    { message }

    Error Response

    { err }
